# Nutrivision-AI-Food-Nutrition-Analyzer
Nutrivision is a full-stack web application that analyzes food images and provides nutritional insights such as calories, protein content, health score, and age-based food recommendations.
The project focuses on practical full-stack development with basic AI/ML logic, making it scalable and easy to understand.

Key Features : 
- Upload food images
- Detect food type (basic AI / dataset-based)
- Estimate calories & protein
- Health rating (Healthy / Moderate / Avoid)
- Age-based food recommendation
- Full-stack architecture (Frontend + Backend)
