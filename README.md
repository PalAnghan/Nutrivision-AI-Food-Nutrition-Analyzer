Nutrivision is a full-stack web application that analyzes food images and provides nutritional insights such as calories, protein content, daily progress tracking, and health tips.
The project demonstrates practical full-stack development with basic AI logic and a clean, modern UI

Live Features : 
- Upload food images
- AI-based food detection (dataset / logic-based)
- Calorie & protein estimation
- Daily calorie progress bar
- Calories per day chart
- Meal history with pagination
- AI daily health tips
- Weekly / Monthly nutrition reports
- Download nutrition report as PDF
- Skeleton loaders (no blank UI)
- Empty state UI (professional UX)
- Smooth animations & transitions

Tech Stack :

Frontend :
- React.js
- Tailwind CSS
- Chart.js
- JavaScript (ES6+)

Backend :
- Node.js
- Express.js
- MongoDB
- REST APIs

Tools & Libraries :
- jsPDF (PDF generation)
- Fetch API
- Git & GitHub

How It Works :
1. User uploads a food image
2. Backend detects food type using logic / dataset
3. Nutrition values are calculated (calories, protein)
4. Data is stored in MongoDB
5. Frontend displays:
  - Daily progress
  - Charts
  - History cards
  - Reports & insights
